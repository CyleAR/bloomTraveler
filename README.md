<h1 align="center">🌸 Bloom Traveler</h1>

**Bloom Traveler**는 탈옥하지 않은 iOS 기기(iPhone, iPad)의 GPS 위치를 PC에서 손쉽게 변경하고 시뮬레이션할 수 있는 Windows용 프로그램입니다.

직관적인 지도 인터페이스를 통해 목적지로 순간이동하거나, 경로를 지정하여 걷는 속도로 이동할 수 있으며, 키보드(WASD)를 이용한 조작도 지원합니다.

---

## ✨ 주요 기능

* **🗺️ 직관적인 지도 UI**: 마우스 클릭만으로 목적지와 경유지를 쉽게 설정할 수 있습니다.
* **🕹️ 조이스틱 컨트롤**: 키보드의 `WASD` 또는 `방향키`를 사용하여 실시간으로 캐릭터를 부드럽게 이동시킬 수 있습니다.
* **🚶‍♂️ 다중 경유지 자동 걷기**: 여러 개의 경유지를 설정하고 원하는 속도(km/h)로 자동 보행이 가능합니다.
* **🚀 스마트 순간이동 (Teleport)**: 50km 이상의 장거리 텔레포트 시, 기기의 위치 캐시를 자동으로 초기화하여 오류(소프트밴 등)를 방지합니다.
* **💓 고무줄 현상 방지 (Heartbeat Anchor)**: 기기의 원래 GPS 신호로 튕기는 현상을 막기 위해 0.5초 간격으로 위치를 강제 고정하는 기능을 지원합니다.
* **🛠️ 숨겨진 개발자 모드 해금**: iOS 16 이상에서 숨겨진 '개발자 모드'를 기기 연결 즉시 자동으로 뚫어줍니다. (기기의 잠금 임시 해제 필요, 개발자 모드 수동으로 활성화 필요)
* **🔄 자동 업데이트 알림**: 새로운 릴리즈가 배포되면 프로그램 시작 시 팝업으로 자동으로 알려줍니다.

## ⚙️ 사전 준비

이 프로그램을 실행하기 위해서는 다음 환경이 필요합니다.

*   **OS**: Windows 10 또는 11 (관리자 권한 필요)
*   **개발자 모드(Developer Mode)가 활성화된 iOS / iPadOS  기기**
*   **Apple 드라이버**: iTunes가 설치되어 있거나, Apple Mobile Device Support 드라이버가 필요합니다. 마이크로소프트 스토어에서 **[Apple 기기 (Apple Devices)]** 앱을 설치하거나 **[iTunes]** 를 설치하세요.
*   **Python**: 3.8 이상 (소스 코드로 실행 시)

## 🎮 사용 방법

- 핫스팟을 끈 기기를 PC에 연결하고 `Bloom Traveler.exe`를 실행합니다. (우측 패널에 초록색 🟢 연결됨 마크 확인)
- **지도 조작:**
   * `좌클릭`: 목적지 설정 (빨간 핀)
   * `휠클릭(가운데 버튼)`: 경유지 추가 (노란 핀, 최대 15개)
   * `우클릭`: 좌표 복사 및 순간이동 팝업 메뉴
- 원하는 **이동 속도(km/h)**를 슬라이더로 조절합니다. (최대 50km/h)
- **[걷기 시작]** 버튼을 누르거나 **WASD or 방향키**를 눌러 이동을 시작합니다.
- 혹은 좌표 입력 창에 원하는 좌표를 입력 후 **좌표로 이동** 버튼을 눌러 이동합니다.

⚠️ **[중요] 200 OK 확인 및 멈춤 해결법:**
   * 지도에서 파란 점이 움직이더라도, 배경에 켜져 있는 까만 콘솔(cmd) 창에 초록색 글씨로 **`200 OK`** 로그가 나타나야만 실제 기기 이동이 완료된 것입니다. 
   * **만약 로그가 뜨지 않고 콘솔 화면이 멈춰있다면, 콘솔 창을 마우스로 한 번 클릭한 뒤 키보드의 `엔터(Enter)` 키를 눌러주세요.** (윈도우 터미널의 화면 일시정지 현상이 풀리면서 이동이 정상적으로 진행됩니다.)

## 🚨 문제 해결 및 팁

### 1. 개발자 모드가 안 켜져요!
애플의 보안 정책상, 아이폰/아이패드에 **화면 잠금 비밀번호(Face ID 포함)**가 걸려있으면 컴퓨터로 개발자 모드를 켤 수 없습니다.
* 기기의 `[설정] -> [Face ID 및 암호]`에서 **임시로 암호를 끄고** 프로그램을 다시 켜보세요. `[개인정보 보호 및 보안]` 맨 아래에 개발자 모드가 나타납니다. 개발자 모드를 켜고 폰이 재부팅된 후에는 암호를 다시 설정하셔도 됩니다.

### 2. 가만히 있는데 원래 위치(현실 위치)로 자꾸 튕겨요! (고무줄 현상)
iOS는 GPS뿐만 아니라 주변 Wi-Fi 신호와 기지국을 읽어 위치를 현실로 강제 보정하려는 성질이 있습니다.
* Bloom Traveler 우측 패널의 **`[💓 고무줄 방지 (강제 하트비트)]`** 체크박스를 켭니다.

### 3. 프로그램에서 내 현재 위치를 모르겠어요!
프로그램을 사용하다 보면 내 현재 위치 (파란색 점)을 지도에서 찾기 어려울 때가 있습니다.
* **걷기 모드 중이 아닐 때**, WASD키나 방향키를 눌러 이동하면 파란색 점이 지도 가운데로 이동합니다.
