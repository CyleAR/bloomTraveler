<h1 align="center">BLOOM TRAVELER</h1>
<h2 align="center">A LightWeight GPS Spoofer for IOS / IpadOS</h2>

**Bloom Traveler**는 iOS 및 iPadOS 기기에서 간편하게 GPS 위치를 시뮬레이션할 수 있도록 설계된 가볍고 효율적인 도구입니다.

## ✨ 주요 특징

- **iOS/iPadOS 지원:** 최신 Apple 모바일 기기와의 호환성을 목표로 합니다.
- **간편한 설정:** 복잡한 과정 없이 위치 정보를 변경할 수 있습니다.
- **Python 기반:** 확장성이 뛰어나며 PyInstaller를 통해 독립 실행형 파일로 빌드 가능합니다.

## 🚀 시작하기

### 요구 사항

- **Python 3.8+**
- iOS/iPadOS 기기 및 USB 연결 케이블
- (선택 사항) 위치 조작을 위한 관련 드라이버 및 라이브러리

### 설치 방법

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/your-username/bloomTraveler.git
   cd bloomTraveler
   ```

2. 가상 환경을 생성하고 활성화합니다:
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # Windows: venv\Scripts\activate
   ```

3. 필요한 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
